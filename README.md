# Todo App（ログイン機能付き）

## 概要
ユーザー登録・ログイン機能付きのToDo管理アプリです。  
Webアプリ開発の基礎（CRUD・認証・DB操作）の理解を目的に作成しました。

---

## 使用技術
- HTML / CSS
- JavaScript


---

## 機能一覧
- ユーザー登録 / ログイン
- ToDo追加
- ToDo一覧表示
- ToDo削除
- 期限設定
- 期限切れの色分け表示

---

## 工夫した点
- セッション管理でユーザーごとにデータ分離
- 期限による視覚的な状態管理
- API設計（フロントとバックエンド分離）

---

## 苦労した点
- ログイン処理とセッション管理
- フロントとバックエンドの連携

---

## 起動方法
npm install  
node server.js  
http://localhost:3000
